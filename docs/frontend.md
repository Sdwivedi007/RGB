# Frontend UX Notes

## Screens
1. **Login** → Auth + role handling
2. **Home Map** → All grounds per client; click opens ground analytics
3. **Ground Analytics** → ortho + AOI + indices + stats
4. **Compare** → A/B split or swipe + opacity
5. **Export** → report history + download
6. **3D View** → Potree viewer

## Map Tools
- Draw: point, line, polygon, circle
- Edit/delete AOI
- Measure: distance, area, angle
- Layer switcher + opacity slider

## Suggested UI Components
- **Left panel**: Layers + indices list + compare controls
- **Right panel**: AOI list + stats + export buttons
- **Bottom**: timeseries or histogram (optional)

## Compare Modes
- **Opacity**: slider to fade layer B
- **Swipe**: draggable vertical divider
- **Split**: screen halves
